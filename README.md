# Personal Portfolio of Karam Almnajed

This repository hosts my personal portfolio, built to present my work in AI, analytics, automation, and decision systems. It includes professional outcomes, selected projects, and credentials. The site is deployed through GitHub Pages and structured for simple customization.

## Use This Portfolio as Your Own

You are free to:

- Clone or fork this project
- Replace the content, style, branding, or structure
- Use tools like Cursor to rewrite sections or generate components
- Adapt it into your own personal portfolio

Everything here is open for reuse.

## Important Notes Before Using

### 1. Remove My Google Verification Files

These files are used only for my domain indexing.
Delete them so you can add your own:

- `google5d9348ad4ab467143.html`
- Any future Google verification files that appear in the repo

After deleting them:
Add your own verification through Google Search Console so your portfolio is indexed and discoverable.

### 2. Replace All Branding Icons That Contain My Initials

This site contains favicon and touch-icon files with my initials embedded in the images.

If you want your own branding, delete and replace all icons, including:

- `apple-touch-icon.png`
- `favicon.ico`
- `favicon.png`
- `favicon-16x16.png`
- `favicon-32x32.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- `site.webmanifest` (update icon paths after replacing)

These icons exist in different sizes so the website displays correctly on all devices.
If you add new icons later, ensure you include multiple resolutions as well.

### 3. Replace My URLs and Contact Links

Update all links in the site, such as:

- Email
- LinkedIn
- GitHub
- WhatsApp
- Any external buttons or footer links

Search for them in:
`index.html`, `main.js`, and any components you modify.

## Running Locally

3. Customize content and styling as needed
